# Privacy Policy

`ad1-core` and `ad1-forensic` are offline Rust libraries. They read AD1 image
files you provide and return parsed structures and forensic findings to your own
program.

- **No data collection.** The libraries do not transmit, upload, or phone home.
- **No network access.** They perform local file I/O only.
- **No telemetry.** No usage analytics are gathered.

Any handling of the evidence you process is governed by your own environment and
policies, not by this software.
